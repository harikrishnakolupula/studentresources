# 🎓 Smart Learning Portal

A full-stack web application designed to provide students with a centralized platform for accessing study materials, programming resources, and interview preparation content. The system enables students to learn efficiently while allowing administrators to manage educational content through a secure dashboard.

---

## 📌 Features

- Student Registration & Login
- Secure Authentication
- Course Management
- Study Materials Upload & Download
- Programming Tutorials
- Interview Preparation Resources
- Responsive User Interface
- Admin Dashboard
- MySQL Database Integration

## 🚀 Installation

### Clone Repository

```bash
git clone https://github.com/yourusername/Smart-Learning-Portal.git
```

### Install Frontend

```bash
cd client
npm install
npm start
```

### Install Backend

```bash
cd server
npm install
npm start
```

---

## ⚙️ Database Setup

1. Install MySQL.
2. Create a database.

```sql
CREATE DATABASE learning_portal;
```

3. Import the SQL file.

```
learning_portal.sql
```

4. Update database credentials in

```
server/config/db.js
```
